A) The Game - top

Monkey Banana Jump is a jumping game where you need to jump and collect as much as banana you can.

With colourful graphics, cute sounds and addictive game play you will get some powers also like apple and watermelon to go high and make your own high score.

This game is build by Construct 2 (.capx File Included)

B) How to play - top

There are two options to play one is by arrows (Left and Right) and another option is to play by TILT mode that will just work on Mobile phones and Tablets.

Use keyboard arrows to move your player or touch on arrows buttons to move

C) Features - top

- HTML 5
- Ludei + Ads Supported
- Playable in all platforms (Mobile, PC, etc.)

- Touch Controls

D) File Includes - top

Included files with this game:

  1. Construct 2 Files
  2. HTML5 Folder
  3. Graphics PNGs

Files Can be edit with construct 2, Notepad and Photoshop

If you'd like to change the main image, open "image"folder, make the necessary adjustments, and then save the file as "relevant.png". Do the same for the images.

If want to edit the construct2 file. You must put the "exporters" folder in to the construct2 root folder,there will be a same folder.

For Reskinning just change pngs in the image folder and you are ready to go.
